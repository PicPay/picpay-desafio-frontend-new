# Recruiter Backend Task Manager

## Description

This project was development for using only in recruiter process dont replic any else place.

## Techonologies

- NestJS (https://nestjs.com/)
- MongoDB (https://www.mongodb.com/)
- Docker (https://www.docker.com/)

## Installation

```bash
$ npm install
```
## Running the app
* For running this app it necessary install Docker see this page (https://www.docker.com/get-started/)

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
